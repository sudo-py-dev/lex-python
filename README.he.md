# 🌐 lex-tg: מנהל הקבוצות האולטימטיבי לטלגרם

[English](README.md) | [עברית](#)

**lex-tg** היא מערכת מודולרית ומתקדמת לניהול קבוצות בטלגרם, המתוכננת לביצועים גבוהים ולאמינות מוחלטת. המערכת עושה שימוש בפרוטוקול **MTProto** ובמחסנית טכנולוגית אסינכרונית ומודרנית, ומעניקה שליטה חסרת תקדים בדינמיקה הקבוצתית תוך דגש על מהירות, תמיכה רב-לשונית ובטיחות סוגים (Type Safety) מחמירה.

---

## ✨ תכונות עיקריות

- 🛡️ **כלי ניהול מתקדמים**: שליטה מלאה על חסימות (Bans), הרחקות (Kicks) והשתקות (Mutes) עם זמני תפוגה מותאמים אישית.
- 🔒 **נעילות דינמיות**: נעילה מיידית של טקסט, מדיה, סטיקרים, קישורים (URLs) ואפילו פקודות בוט.
- ⚡ **עוזר בינה מלאכותית (AI)**: אינטגרציה מובנית עם Google Gemini ו-OpenAI לסיוע חכם בקבוצה ומענה אוטומטי להודעות.
- 🌊 **בקרת הצפה ופשיטות (Flood & Raid)**: מנגנוני הגבלת קצב (Rate-limiting) ונעילה חכמים למניעת ספאם ומתקפות בוטים.
- 🌍 **תמיכה רב-לשונית רחבה**: תרגום מלא ליותר מ-20 שפות, המופעל על ידי מנוע תרגום מרכזי וחכם.
- 🎛️ **פאנל ניהול (Admin Panel) פרימיום**: ממשק ניהול נוח ואינטואיטיבי מבוסס כפתורים (Callbacks) להגדרת הקבוצה בקלות.
- 🧩 **ארכיטקטורת פלאגינים שטוחה**: מערכת פלאגינים יעילה במיוחד המבוססת על קובץ יחיד לכל רכיב, המבטיחה תחזוקה קלה ופשוטה.

## 🛠️ מחסנית טכנולוגית

| שכבה | טכנולוגיה |
| :--- | :--- |
| **לקוח ליבה** | [Pyrogram](https://docs.pyrogram.org/) (Async MTProto) |
| **מסד נתונים** | [SQLAlchemy 2.0](https://www.sqlalchemy.org/) (ORM מודרני עם טיפוסיות חזקה) |
| **מטמון (Cache)** | [AsyncSnapshotCache](src/cache/local_cache.py) (מטמון מקומי מהיר, ללא תלות ב-Redis) |
| **מנהל חבילות** | [uv](https://astral.sh/uv/) (ביצועים קיצוניים ובידוד סביבות עבודה) |
| **בטיחות סוגים** | [Mypy](https://mypy-lang.org/) (אכיפת סוגים קפדנית ללא פשרות) |

## 🚀 התחלה מהירה

### 1. התקנת `uv`
הפרויקט דורש התקנה של `uv` לצורך ניהול תלויות מהיר במיוחד וסביבות וירטואליות מבודדות.
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 2. הגדרת סביבת העבודה
שכפלו את המאגר (Repository) וסנכרנו את התלויות:
```bash
uv sync
cp .env.example .env
```
> [!IMPORTANT]
> הקפידו למלא את הפרטים `API_ID`, `API_HASH`, ו-`BOT_TOKEN` בקובץ ה-`.env`. ניתן גם לעדכן את שם הבוט (`BOT_NAME`) בקובץ `src/config.py` (ברירת המחדל היא `lex-tg`).

### 3. הרצת הבוט
```bash
uv run bot
```
